# Night of the dead
Steam Description  
Build defenses against nightly waves of zombies. Survive and escape the island.

### Server Ports
Night of the dead requires up to 2 ports

Game query port (default 27015)
Game client port (default 7777)

| Port    | default |
|---------|---------|
| Game    | 27015   |
| Client  | 7777    |
